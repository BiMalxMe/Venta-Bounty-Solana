# Solana Wallet Inventory 🪙

A living reference for all wallets supporting Solana, detailing their key features and Solana Pay QR-code compatibility.

## 📊 What’s Inside
- ✅ Wallet Inventory (mobile, browser-extension, desktop)
- ✅ Feature Matrix:
  - Platforms (iOS, Android, Chrome, etc.)
  - Custody Model (Self-custody, MPC, Custodial)
  - In-app DEX swap
  - NFT gallery
  - In-app staking
  - Fiat on/off ramps
  - Push-notification support
  - Solana Pay QR reader (Yes / Partial / No)
- ✅ Version tested + Date of testing
- ✅ Wallet logos included
- ✅ Interactive Dashboard (Notion)
- ✅ Raw data (JSON, CSV)

## 🔄 Update Instructions
1. Open `wallets.json`
2. Add new wallets to the `"wallets"` array
3. Update `version_tested` and `test_date`
4. Push changes or upload to your Notion dashboard

## 📸 Evidence
Screenshots and recordings verifying Solana Pay behaviour are included in `/evidence`.
